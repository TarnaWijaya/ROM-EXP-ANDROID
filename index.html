• Cara Pasang Script Dengan Mudah

Cara Pasang Mudah ditermux

1. Folder ini harus dimemory perangkat

2. install termux di website ini: Unduh Termux

3. buka termux

4. izinkan semua fitur diaplikasi termux

5. salin script ini:

___1. " termux-setup-storage "

___2. " cd /storage/emulated/0/RomExp "

___3. " chmod +x RomExp.sh "

___4. " ./RomExp.sh "

___5. " pkg install bash "
